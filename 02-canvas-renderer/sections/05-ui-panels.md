# 05 — UI Panels

## Summary

Port the four DOM-based UI components from index.html: category legend, search bar with results, node info panel, and stats bar. Each is a self-contained class that creates its DOM elements inside the container div, updates reactively, and cleans up on destroy.

Key behavior from index.html:

All styles are applied inline or via a `<style>` element injected into the container — no global stylesheet dependency. Use `bp-` prefix on all CSS classes to avoid collisions.

### Shared Style Injection

All panels inject styles via a single `<style>` element appended to the container on first construction:

```typescript
function injectStyles(container: HTMLDivElement, theme: ThemeColors): void {
  if (container.querySelector('style[data-bp-styles]')) return;
  const style = document.createElement('style');
  style.setAttribute('data-bp-styles', '');
  style.textContent = `
    .bp-legend { position: absolute; top: 12px; right: 12px; ... }
    .bp-search { position: absolute; top: 12px; left: 12px; ... }
    .bp-info-panel { position: absolute; top: 50%; right: 12px; ... }
    .bp-stats { position: absolute; bottom: 12px; left: 50%; ... }
    /* All colors from ThemeColors, not hardcoded */
  `;
  container.appendChild(style);
}
```

On theme change, the style element is replaced with updated color values.
